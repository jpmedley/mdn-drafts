The operator read-only property of the {{domxref("CSSMathValue")}} interface returns a string with the name of an operator specific to the subtype it is called on. It will always be one of "sum", "product", "negate", "invert", "min", or "max".

Syntax

var operator = cssMathValueSubtype.operator

Value

one of "sum", "product", "negate", "invert", "min", or "max".

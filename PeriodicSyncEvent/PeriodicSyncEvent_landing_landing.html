The PeriodicSyncEvent interface of the Web Periodic Background Synchronization APIprovides a way to run tasks in the service worker with network connectivity. An instance of this event is passed to the {{domxref('ServiceWorkerGlobalScope.onperiodicsync')}} handler. This happens periodically, at an interval greater than or equal to that set in {{domxref('PeriodicSyncManager.register()')}}. Other implementation-specific factors such as the user's engagement with the site decide the actual interval.

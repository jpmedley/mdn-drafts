The read-only types property of the {{domxref("ClipboardItem")}} interface returns an {{jsxref("Array")}} of {{Glossary("MIME type"), 'MIME types'}} available within the {{domxref("ClipboardItem")}}

Syntax

var types = clipboardItem.types;

Value

An {{jsxref("Array")}} of available {{Glossary("MIME type"), 'MIME types'}}.

Examples

In the below example, we're returning all items on the clipboard via the {{domxref("clipboard.read()")}} method. Then checking the types property for available types before utilizing the {{domxref("ClipboardItem.getType()")}} method to return the {{domxref("Blob")}} object. If no clipboards contents is found for the specified type, an error is returned.

async function getClipboardContents() {
  try {
    const clipboardItems = await navigator.clipboard.read();

    for (const clipboardItem of clipboardItems) {
      
      for (const type of clipboardItem.types) {
        const blob = await clipboardItem.getType(type);
        // we can now use blob here
      }

    }

  } catch (err) {
    console.error(err.name, err.message);
  }
}

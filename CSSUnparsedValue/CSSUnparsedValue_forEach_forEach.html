The CSSUnparsedValue.forEach() method executes a provided function once for each element of {{domxref('CSSUnparsedValue')}}.

Syntax

CSSUnparsedValue.forEach(function callback(currentValue[, index[, array]]) {
    //your iterator
}[, thisArg]);

Parameters

callback
The function to execute for each element, taking three arguments:
currentValue
The value of the current element being processed.
index{{optional_inline}}
The index of the current element being processed.
array{{optional_inline}}
The CSSUnparsedValue thatforEach() is being called on.
thisArg {{Optional_inline}}

Value to use as this (i.e the reference Object) when executing callback.

Return value

{{jsxref("undefined")}}.

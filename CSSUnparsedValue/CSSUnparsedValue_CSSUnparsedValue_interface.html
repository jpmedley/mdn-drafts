The CSSUnparsedValue interface of the the CSS Typed Object Model API represents property values that reference custom properties. They are composed of a list of string fragments and variable references.

Constructor

{{domxref("CSSUnparsedValue.CSSUnparsedValue()")}}
Creates a new CSSUnparsedValue object.

Properties

Methods

Inherits methods from {{domxref('CSSStyleValue')}}.

{{domxref('Widget.entries()')}}
Returns an array of a given object's own enumerable property [key, value] pairs, in the same order as that provided by a {{jsxref("Statements/for...in", "for...in")}} loop (the difference being that a for-in loop enumerates properties in the prototype chain as well).
{{domxref('Widget.forEach()')}}
Executes a provided function once for each element of {{domxref('CSSUnparsedValue')}}.
{{domxref('Widget.keys()')}}
Returns a new Array Iterator object that contains the keys for each index in the array.
{{domxref('Widget.values()')}}
Returns a new Array Iterator object that contains the values for each index in the CSSUnparsedValue object.

The add() method of the {{domxref("CSSNumericValue")}} interface adds one or more unit-specific numbers and returns the result as a {{domxref('CSSUnitValue')}} or null if untyped values are passed. This method does not change the initial value.

Syntax

var cssUnitValue = cssNumericValueChildInstance.sum([values])

Parameters

values
One or more unit-specific values to add to the intitial value.

Return value

A {{domxref('CSSNumbericValue')}}.

Example

The following examples illustrate three facets of the add() method's behavior.

var pxs = CSS.px(1);
var newVal = pxs.add(CSS.px(2), CSS,px(2));

// Prints {value: 1, unit: "px"}
console.log(pxs);

// Prints {value: 5, unit: "px"}
console.log(newVal)

var noVAl = pxs.add(2)
// Prints null
console.log(noVal)

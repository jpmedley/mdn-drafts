The max() method of the {{domxref("CSSNumericValue")}} interface returns the higher of two numbers or null if the two numbers are not the same type. This method does not change the initial value.

Syntax

var cssUnitValue = cssNumericValueChildInstance.max([values])

Parameters

values
One or more unit-specific values to add to the intitial value.

Return value

A {{domxref('CSSNumbericValue')}}.

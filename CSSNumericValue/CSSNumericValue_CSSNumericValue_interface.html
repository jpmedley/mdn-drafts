The CSSNumericValue interface of the the CSS Typed Object Model API is a base class for interfaces provideing mathematical operations on CSS values.

Interfaces based on CSSNumericValue

Below is a list of interfaces based on the CSSNumericValue interface.

  • {{domxref('CSSMathVAlue')}}
  • {{domxref('CSSUnitVAlue')}}

Properties

None.

Methods

{{domxref('CSSNumericValue.add()')}}
Adds one or more unit-specific numbers and returns the result as a {{domxref('CSSUnitValue')}} or null if untyped values are passed.
{{domxref('CSSNumericValue.div()')}}
Divides one or more numbers and returns the result or a {{domxref('CSSMathProduct')}} if values other than numbers are passed
{{domxref('CSSNumericValue.equals()')}}
Tests the equality of the provided numbers and returns the result.
{{domxref('CSSNumericValue.max()')}}
Returns the higher of two numbers or null if the two numbers are not the same type.
{{domxref('CSSNumericValue.min()')}}
Returns the lower of two numbers or null if the two numbers are not the same type.
{{domxref('CSSNumericValue.mul()')}}
Multiplies one or more numbers and returns the result or a {{domxref('CSSMathProduct')}} if values other than numbers are passed.
{{domxref('CSSNumericValue.parse()')}}
Converts the supplied digit-containg string to a {{domxref('CSSUnitValue')}}.
{{domxref('CSSNumericValue.sub()')}}
Subtracts one or more unit-specific numbers and returns the result as a {{domxref('CSSUnitValue')}} or null if untyped values are passed. This method does not change the initial value.
{{domxref('CSSNumericValue.to()')}}
Converts the current CSSNumericValue to the provided units.
{{domxref('CSSNumericValue.toSum()')}}
Returns a {{domxref('CSSMathSum')}} object whose members are the original value converted to the specified types.
{{domxref('CSSNumericValue.type()')}}
Returns the base type of a CSSNumericValue and an associated percent hint.

The div() method of the {{domxref("CSSNumericValue")}} interface divides one or more numbers and returns the result or a {{domxref('CSSMathProduct')}} if values other than numbers are passed. This method does not change the initial value.

Syntax

var cssUnitValue = cssNumericValueChildInstance.div([values])

Parameters

values
One or more unit-specific values to add to the intitial value.

Return value

A {{domxref('CSSNumbericValue')}} or a {{domxref('CSSMathProduct')}} if anything other than a number is passed.

Example

var newDeg = CSS.deg(180).mul(2, 2)
console.log(newDeg);
// Prints {value: 45, unit: "deg"}
console.log(newDeg.toString())
// Prints 45deg

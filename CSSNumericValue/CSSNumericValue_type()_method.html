The type() method of the {{domxref("CSSNumericValue")}} interface returns the base type of a CSSNumericValue and an associated percent hint.

Syntax

var cssNumericType = cssNumericValueChildInstance.type()

Parameters

None.

Return value

One of the following {type: percentHint} maps:

  • {} (Indicates a number.)
  • {angle: 1}
  • {flex: 1}
  • {Frequency: 1}
  • {length: 1}
  • {percent: 1}
  • {resolution: 1}
  • {time: 1}

The parse() method of the {{domxref("CSSNumericValue")}} interface converts the digit-containg string to a {{domxref('CSSUnitValue')}}.

Syntax

var cssNumericValue = cssNumericValueChildInstance.([cssText]).

Parameters

cssText
desc

Return value

A {{domxref('CSSNumbericValue')}}.

Example

var parsed = CSSNumericValue.parse('42px');

// Prints {value: 42, unit: "px"}.
console.log(parsed)
var parsed2 = CSSNumericValue.parse('I am 42'); // Throws an error. console.log(parsed2);

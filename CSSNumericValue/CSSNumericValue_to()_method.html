The to() method of the {{domxref("CSSNumericValue")}} interface converts the current CSSNumericValue to the provided units.

Syntax

var cssUnitValue = cssNumericValueChildInstance.to(unit)

Parameters

unit
A string representing the units to convert to. Valid values are:
  • number
  • percent

Length values

  • ch
  • cm
  • em
  • ex
  • ic
  • in
  • lh
  • mm
  • pc
  • pt
  • px
  • Q
  • rem
  • rlh
  • vb
  • vh
  • vi
  • vmax
  • vmin
  • vw

Angle values

  • deg
  • grad
  • rad
  • turn

Time values

  • ms
  • s

Frequency values

  • Hz
  • kHz

Resolution values

  • dpi
  • dpcm
  • dppx

Flex

  • fr

Return value

A {{jsxref('CSSUnitValue')}}.

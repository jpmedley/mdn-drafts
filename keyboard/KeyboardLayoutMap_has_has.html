The KeyboardLayoutMap.has() method returns a boolean indicating whether the KeyboardLayoutMap object has an element with the specified key.

Syntax

var aBoolean = KeyboardLayoutMap.has(key)

Parameters

key
The key of an element to search for in the map.

Return value

A {{jsxref('Boolean')}} indicating whether the specifed key was found.

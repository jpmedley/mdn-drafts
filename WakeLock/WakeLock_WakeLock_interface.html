The WakeLock interface implements the {{domxref('Screen Wake Lock API')}}, allowing a document to acquire a screen wake lock. It provides a way to prevent device screens from dimming or locking when an application needs to keep running.

The system wake lock is exposed through the global {{domxref('Navigator.wakelock')}} property.

Methods

request()
Requests a {{domxref("WakeLockSentinel")}} object, which returns a {{domxref("Promise")}} that resolves with a {{domxref("WakeLockSentinel")}} object.

Examples

The following asynchronous function requests a {{domxref("WakeLockSentinel")}} object. The {{domxref("WakeLock.request")}} method is wrapped in a try...catch statement to account for if the browser refuses the request for any reason.

const requestWakeLock = async () => {
  try {

    const wakeLock = await navigator.wakeLock.request('screen');

  } catch (err) {
    // the wake lock request fails - usually system related, such low as battery

    console.log(`${err.name}, ${err.message}`);
  }
}

requestWakeLock();

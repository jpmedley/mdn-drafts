The request() method of the {{domxref("WakeLock")}} interface returns a {{jsref("Promise")}} that when resolved returns a {{domxref("WakeLockSentinel")}} object, which in turn allows control over screen dimming and locking.

Syntax

var wakeLock = navigator.wakeLock.request('screen');

Parameters

type
A valid wake lock type
The different types of Wake Lock available
type Description
screen Prevents devices from dimming or locking the screen

Return value

A {{domxref("Promise")}} that resolves with a {{domxref("WakeLockSentinel")}} object.

Exceptions

NotAllowedError
The user agent denies the wake lock request (due to system settings or low battery). Or the document does not have a browser context, is not fully active or is not visible (ie not the active tab.

Examples

The following asynchronous function requests a {{domxref("WakeLockSentinel")}} object. The {{domxref("WakeLock.request")}} method is wrapped in a try...catch statement to account for if the browser refuses the request for any reason.

const requestWakeLock = async () => {
  try {

    const wakeLock = await navigator.wakeLock.request('screen');

  } catch (err) {
    // the wake lock request fails - usually system related, such low as battery

    console.log(`${err.name}, ${err.message}`);
  }
}

requestWakeLock();

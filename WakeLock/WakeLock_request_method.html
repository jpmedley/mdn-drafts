The request() method of the {{domxref("WakeLock")}} interface returns a {{jsxref("Promise")}} that resolves with a {{domxref("WakeLockSentinel")}} object, which allows control over screen dimming and locking.

Syntax

var wakeLock = navigator.wakeLock.request(type);

Parameters

type
Options are as follows:
  • 'screen': Requests a screen wake lock. Prevents devices from dimming or locking the screen.

Return value

A {{domxref("Promise")}} that resolves with a {{domxref("WakeLockSentinel")}} object.

Exceptions

NotAllowedError
The user agent denies the wake lock request due to system settings or low battery. Or the document does not have a browser context, is not fully active or is not visible (i.e. not the active tab).

Examples

The following asynchronous function requests a {{domxref("WakeLockSentinel")}} object. The {{domxref("WakeLock.request")}} method is wrapped in a try...catch statement to account for if the browser refuses the request for any reason.

const requestWakeLock = async () => {
  try {

    const wakeLock = await navigator.wakeLock.request('screen');

  } catch (err) {
    // the wake lock request fails - usually system related, such low as battery

    console.log(`${err.name}, ${err.message}`);
  }
}

requestWakeLock();
